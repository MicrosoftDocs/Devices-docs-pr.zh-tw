## <span data-ttu-id="79a6b-108">表面驅動程式與固件支援週期</span><span class="sxs-lookup"><span data-stu-id="79a6b-108">Surface driver and firmware support lifecycle</span></span>
 
<span data-ttu-id="79a6b-109">表面驅動程式和固件週期由兩個部分組成：裝置的驅動程式和固件支援週期，以及在該期間內提供給 OS 版本的支援。</span><span class="sxs-lookup"><span data-stu-id="79a6b-109">The Surface Driver and Firmware Lifecycle is composed of two parts: The driver and firmware support period for a device and the support provided for OS versions during that period.</span></span>

- **<span data-ttu-id="79a6b-110">裝置支援期間。</span><span class="sxs-lookup"><span data-stu-id="79a6b-110">Device support period.</span></span>** <span data-ttu-id="79a6b-111">裝置支援期間定義 Surface 將支援裝置的驅動程式和固件更新期間。</span><span class="sxs-lookup"><span data-stu-id="79a6b-111">The device support period defines the period during which Surface will support driver and firmware updates for a device.</span></span> <span data-ttu-id="79a6b-112">裝置支援期間會在裝置發行時開始。</span><span class="sxs-lookup"><span data-stu-id="79a6b-112">The Device Support Period starts when a device is released.</span></span> <span data-ttu-id="79a6b-113">在第一次發行裝置後，Surface 裝置將會收到至少4年的驅動程式和固件更新。</span><span class="sxs-lookup"><span data-stu-id="79a6b-113">Surface devices will receive driver and firmware updates for at least 4 years from when the device was first released.</span></span> <span data-ttu-id="79a6b-114">在支援期間長於4年的情況下，系統會在上次服務的日期之前發佈更新結束的服務日期。</span><span class="sxs-lookup"><span data-stu-id="79a6b-114">In cases where the support duration is longer than 4 years, an updated end of servicing date will be published in advance of the date of last servicing.</span></span>

- **<span data-ttu-id="79a6b-115">作業系統版本支援。</span><span class="sxs-lookup"><span data-stu-id="79a6b-115">OS version support.</span></span>** <span data-ttu-id="79a6b-116">OS 版本支援定義在裝置支援期間內，Surface 支援的作業系統版本。</span><span class="sxs-lookup"><span data-stu-id="79a6b-116">OS version support defines the operating system versions supported by Surface during the device support period.</span></span> <span data-ttu-id="79a6b-117">在前30個月發行的 Windows 作業系統版本中，Surface 裝置將會收到驅動程式和固件更新。</span><span class="sxs-lookup"><span data-stu-id="79a6b-117">Surface devices will receive driver and firmware updates for Windows OS versions released in the prior 30 months.</span></span> <span data-ttu-id="79a6b-118">Surface 不支援低於裝置版本支援之 OS 版本的 Windows 作業系統版本。</span><span class="sxs-lookup"><span data-stu-id="79a6b-118">Surface will not support Windows OS versions earlier than the OS versions supported at device release.</span></span> <span data-ttu-id="79a6b-119">如需 Surface 裝置支援的最低作業系統版本，請參閱支援 [surface 的作業系統。](https://support.microsoft.com/help/2858199/surface-supported-operating-systems)</span><span class="sxs-lookup"><span data-stu-id="79a6b-119">For the minimum supported OS version of your Surface device, see [Surface supported operating systems support.](https://support.microsoft.com/help/2858199/surface-supported-operating-systems)</span></span>  

 
<span data-ttu-id="79a6b-120">在服務日期結束後，裝置支援週期結束後，裝置會依照  [Microsoft 週期原則支援頁面](https://support.microsoft.com/hub/4095338/microsoft-lifecycle-policy)上的說明，繼續接收 windows 作業系統功能和安全性更新。</span><span class="sxs-lookup"><span data-stu-id="79a6b-120">Once the device support period is concluded at the End of Servicing Date, devices will continue to receive Windows OS feature and security updates in accordance with the Windows Lifecycle Policy as described on the  [Microsoft Lifecycle Policy support page](https://support.microsoft.com/hub/4095338/microsoft-lifecycle-policy).</span></span>
