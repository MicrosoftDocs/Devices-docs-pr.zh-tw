## <span data-ttu-id="56bb5-198">範例</span><span class="sxs-lookup"><span data-stu-id="56bb5-198">Examples</span></span> 

**<span data-ttu-id="56bb5-199">使用 PowerShell 來檢索 SKU</span><span class="sxs-lookup"><span data-stu-id="56bb5-199">Retrieving the SKU by using PowerShell</span></span>**
