# <span data-ttu-id="60179-104">使用 Intune 管理 Surface Hub 2S</span><span class="sxs-lookup"><span data-stu-id="60179-104">Manage Surface Hub 2S with Intune</span></span>

## <span data-ttu-id="60179-105">使用 Intune 註冊 Surface Hub 2S</span><span class="sxs-lookup"><span data-stu-id="60179-105">Register Surface Hub 2S with Intune</span></span>

<span data-ttu-id="60179-106">Surface Hub 2S 可讓 IT 系統管理員使用行動裝置管理（MDM）提供者管理設定和原則。</span><span class="sxs-lookup"><span data-stu-id="60179-106">Surface Hub 2S allows IT administrators to manage settings and policies using a mobile device management (MDM) provider.</span></span> <span data-ttu-id="60179-107">Surface Hub 2S 具備內建的管理元件，可與管理伺服器溝通，因此不需要在裝置上安裝其他用戶端。</span><span class="sxs-lookup"><span data-stu-id="60179-107">Surface Hub 2S has a built-in management component to communicate with the management server, so there is no need to install additional clients on the device.</span></span>

### <span data-ttu-id="60179-108">手動註冊</span><span class="sxs-lookup"><span data-stu-id="60179-108">Manual registration</span></span>

1. <span data-ttu-id="60179-109">請以 Surface Hub 2S 的本機系統管理員身分登入，然後開啟 **[設定]** 應用程式。</span><span class="sxs-lookup"><span data-stu-id="60179-109">Sign in as a local administrator on Surface Hub 2S and open the **Settings** app.</span></span> <span data-ttu-id="60179-110">點選 **Surface Hub** > \*\* [裝置管理]\*\*，之後選擇 **+** 新增。</span><span class="sxs-lookup"><span data-stu-id="60179-110">Select **Surface Hub** > **Device management** and then select **+** to add.</span></span>

   ![使用 Intune 註冊 Surface Hub 2S](images/sh2-set-intune1.png)<br>

### <span data-ttu-id="60179-113">自動註冊－ Azure Active Directory 附屬連結</span><span class="sxs-lookup"><span data-stu-id="60179-113">Auto registration — Azure Active Directory Affiliated</span></span>

<span data-ttu-id="60179-114">在初始設定過程中，在將 Surface Hub 連結至一個已啟用 Intune 自動註冊功能的 Azure AD 租用者帳號時，裝置便會進行 Intune 的自動註冊程序。</span><span class="sxs-lookup"><span data-stu-id="60179-114">During the initial setup process, when affiliating a Surface Hub with an Azure AD tenant that has Intune auto enrollment enabled, the device will automatically enroll with Intune.</span></span> <span data-ttu-id="60179-115">如需更多詳細資訊，可參照 [ Windows 裝置適用的 Intune 註冊方法](https://docs.microsoft.com/intune/enrollment/windows-enrollment-methods)。</span><span class="sxs-lookup"><span data-stu-id="60179-115">For more information, refer to [Intune enrollment methods for Windows devices](https://docs.microsoft.com/intune/enrollment/windows-enrollment-methods).</span></span> <span data-ttu-id="60179-116">Surface Hub 要成為 Intune 的「相容裝置」，則必須具有 Azure AD 的附屬連結和 Intune 的自動註冊功能。</span><span class="sxs-lookup"><span data-stu-id="60179-116">Azure AD affiliation and Intune auto enrollment is required for the Surface Hub to be a "compliant device" in Intune.</span></span>
