# <a name="install-and-mount-surface-hub-2s-85"></a>安裝和安裝 Surface Hub 2S 85"

本文提供在商業環境中實際安裝 Microsoft Surface Hub 2S 85" 的支援指引。 

## <a name="unboxing-video"></a>取消文字方塊影片

目前提供如何移動及安裝 Surface Hub 2S 85 的影片。 開始之前，請先查看 [Microsoft Surface Hub 2S 85" 取消框線及設定](https://www.microsoft.com/zh-cn/videoplayer/embed/RE4MRqV)。  

## <a name="follow-all-safety-precautions"></a>遵循所有安全預防措施 

> [!WARNING]
> **處理與網站準備**<br>
> - 裝置非常重。 若要降低因裝置大小與重量而造成個人傷害、死亡或損壞的風險，請保持裝置直立。
>- 在將裝置移至安裝裝置的位置之前，請調查網站，以決定如何安全地將裝置移至將裝置解壓縮並安裝的位置。
>- 請一直使用至少兩個人來解壓縮和安裝。
>- 裝置一旦解壓縮後，應該立即安裝，所以在解壓縮之前，安裝系統應該就位。 如果您要將裝置安裝在滾動支架上，請鎖定或封鎖支架的滑杆，然後再進行開箱。
>- 為了避免發生脫扣危險，請保持元件區沒有包裝材料。

> [!IMPORTANT]
>繼續進行之前，請審查下列附件中列出的其他 [安全](#appendix) 資訊。

## <a name="1-remove-outer-packaging-materials"></a>1. 移除外包裝材料

1. 移除及回收外蓋。
2. 剪下四 (4) 塑膠綁帶。
3. 開啟並移除六個 (6 個) 剪輯。
4. 移除外蓋，然後移開及移除前後面板。

 ![移除外包裝](images/hub85mount1a.png) <br>

## <a name="2-remove-black-outer-packaging-foam"></a>2. 移除黑色外包裝包裝

1. 移除黑色角落的 (4) 。
2. 移除黑色中心支線支援。

 ![移除黑色外包裝包裝](images/hub85mount2a.png) <br>

> [!Caution]
> 請勿移除任何白色浮水印、系帶或包裝材料，直到將放置它之購物車或牆掛式裝置旁的 Hub 2S。
其他系帶材料會放在包裝背面手部小盒中。 原始或取代的系帶材料必須在移動前就位，尤其是在旋轉裝置及其保護套件之前
 

## <a name="3-remove-inner-packaging-frame-from-pallet"></a>3. 移除包裝盒的內部包裝框

1. 將元件元件移至提升層位置。
2. 鎖定滾輪軸向 (4) 。
3. 將內部包裝從包裝上滑開。

 ![移除包裝盒的內部包裝框](images/hub85mount3b.png) <br>

## <a name="4-rotate-packaging-frame-to-fit-elevator"></a>4. 旋轉包裝框架以容納乘機

1. 旋轉框架以容納乘用卡
2. 旋轉內部框架的斜面端，然後滑入乘風車。

 ![旋轉包裝框架以容納乘機](images/hub85mount4a.png) <br>

> [!TIP]
> 轉輪元件是自訂配合內部框架包裝封裝量的大小，可在整個安裝網站傳遞過程中使用。 內部木框架尾片有滑過板。
 
## <a name="5-remove-from-elevator"></a>5. 從提升層移除

1. 從高樓滑出
2. 鎖定 x4 軸 (輪軸) 。

 ![從乘機移除](images/hub85mount5a.png) <br>


## <a name="6-place-surface-hub-85-back-on-pallet"></a>6. 將 Surface Hub 85" 放回系統

1. 使用斜面端，旋轉重播大軸。
2. 將滾輪軸向 (x4) 。
3. 將產品移至牆面或購物車安裝的位置。

 ![將 Surface Hub 85" 放回系統](images/hub85mount6a.png) <br>

## <a name="7-place-surface-hub-85-on-wall-mount-or-cart"></a>7. 將 Surface Hub 85" 放在掛式裝置或購物車上

1. 將中心 2S 放在牆面安裝或購物車的前方。
2. 鎖定 x4 軸 (輪軸) 。
3. 將內部包裝從包裝上滑開。
4. 剪下 3 個塑膠綁帶。
5. 移除外蓋。
6. 移除白色水泡部分。
7. 移除歡迎套件。
8. 垂直起重以移除終點。
9. 使用四個手輪固定在樹基座上的樹尾部分來移除。

   ![將 Surface Hub 85" 放在掛式裝置或購物車上](images/hub85mount7a.png) <br>

   > [!CAUTION]
   >  請勿讓 Hub 2S 85" 自動離開。 需要另一個人來固定裝置。 一旦移除尾端，至少需有一個人與 Hub 2S 保持聯絡，直到安裝于牆面安裝或購物車完成為止。

10. 向上和返回將 ply%。請解除 ply以結束部分。
11. 移除背面的凸起標籤。
12. 將 Hub 2S 從下層紙盤上提空，並放在購物車或牆面安裝裝置上。

    > [!NOTE]
    > 按照購物車或牆面安裝製造指示，在將 Hub 2S 從內部包裝盒移除之前，先準備安裝系統。
 
    > [!TIP]
    > 當握住 Hub 2S 時，請注意，手拿著會從較低層開始剪下。 您必須小心處理，不要握住喇叭所在的裝置。 裝置外蓋兩端的圖形提供一般喇叭位置。

13. 在 x2 (中放鬆) 。
14. 移除布蓋。
15. 用於放置 x2 (、) 和電源線的筆的位置。
16. 將筆 (x2) 、相機和電源線。
17. 移除 x4 (上的) 。
18. 按右下角的電源按鈕。 安裝現已完成。

## <a name="appendix"></a>附錄 

### <a name="additional-safety-information"></a>其他安全資訊

> [!WARNING]
> **重物/人體工學提升**<br>
> 裝置非常重。 若要降低與裝置相關的拉傷、死亡或損壞風險，建議您至少讓兩位或多人讓裝置解除。 在起重和/或移動裝置時，使用適當的舉重立體非常重要。
> 使用良好的人體工學提升作業，包括但不限於：
> - 提前規劃。 確定提升小組同意計畫。
>- 判斷您是否可以解除裝置。 是否太重或太不貴？
>- 決定是否需要舉重輔助。
>- 檢查您的環境是否發生障礙和滑滑表面。
>- 用您的雙腳而不是背來提升。
>- 彎曲您的膝部，讓背保持直線。
>- 將單位保持靠近您的身體。
>- 將身體中心在單位上。 讓雙腳與雙背寬度分開。
>- 順暢地向上升空。
>- 保持您的直向;提升時或負載提升後請勿旋轉


> [!WARNING]
>  **適當的安裝**<br> 裝置重且連接到購物車或牆面安裝。 若要降低裝置受傷害、死亡或損毀的風險： 
>- 請遵循購物車或牆面安裝製造商提供的所有指示。
>- 確保建議的安裝系統能支援此裝置重量。
>- 只要使用安裝系統時提供的安裝硬體。
>- 請根據製造商的指示，確保所有的螺杆都安全上鎖。
>- 請勿放開裝置，直到您確定特定裝置與安裝系統附件點完全連接。
>- Microsoft 建議使用專為您的裝置設計的購物車或牆面安裝系統。 對於使用其他安裝系統所造成的任何損害、損壞或死亡，Microsoft 不負任何責任。

> [!WARNING]
>  **牆面或其他安裝表面未出現的風險** <br>
牆面和其他安裝表面可能包含纜線、瓦斯線，以及其他看不見的風險或障礙。 剪下或切入看不見的公文，可能會導致嚴重的個人傷害或死亡。
安裝程式有責任在安裝之前找出未看見的潛在危險，並避免這些危險。 在進行鑽進和/或剪下之前，評估安裝環境，並永遠確保牆面或其他安裝表面沒有未看得到的風險。

> [!WARNING]
> **提示危險**<br>
> 若要避免在移動時造成購物車/支架裝置受傷害、死亡或損毀的風險：
> - 請僅使用此裝置相容的購物車/攤位。
>- 請遵循購物車/支架製造商提供的所有指示，以移動或移轉安裝中的裝置。
>-  請勿從裝置或購物車/支架上當機或放置重物。
>-  在移動購物車/支架安裝裝置之前，請根據需要中斷連接電源線和其他纜線。 移動購物車/獨立安裝裝置時，請小心並慢慢移動。 按照購物車/支架製造商的指示移動或移轉支架。
>-  在上或下斜面運送購物車/支架安裝裝置時，請小心小心。 絕不讓購物車/支架安裝裝置自動留在或靠近斜道。
>-  只有成人應該移動購物車/獨立安裝的裝置。

> [!CAUTION]
> **觸控式螢幕玻璃**<br>
與大部分的觸控式螢幕一樣，裝置上的觸控式螢幕是由玻璃製成。 如果裝置被放下或受到嚴重影響，玻璃可能會中斷。 若要降低造成人員傷害的風險，如果玻璃被損壞、打卡或裂開，請避免觸碰螢幕，並安排更換螢幕。
產品有限保固不涵蓋因不當使用或不當使用裝置所造成的觸控式螢幕損壞或卡住的問題。

> [!WARNING]
>  **適當的安裝**<br>
為了避免與不當裝置安裝相關的危險，安裝必須由在開始作業之前已閱讀並瞭解安裝指示的人執行。 如果您沒有所需的設備或專業知識，或如果您不確定的安裝表面，可以適當支援，請查閱專業安裝程式。
